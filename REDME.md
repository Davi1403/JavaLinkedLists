# Java Linked List Implementations

This repository contains implementations of three types of linked lists in Java:

- **ListaSL** – Singly Linked List  
- **ListaDL** – Doubly Linked List  
- **ListaLC** – Circular Linked List  

## 📁 Folder Structure

JavaLinkedLists/
├── ListaSL/ # Singly Linked List
├── ListaDL/ # Doubly Linked List
└── ListaLC/ # Circular Linked List
